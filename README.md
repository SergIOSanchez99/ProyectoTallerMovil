# Taller Móvil - Proyecto de Segmentación y Clasificación de Cáncer de Colon

## Descripción

Aplicación móvil desarrollada en Flutter para segmentación y clasificación de cáncer de colon. La aplicación incluye una pantalla de login moderna y elegante.

## Características

- Pantalla de login con diseño moderno
- Campos de usuario y contraseña
- Validación básica de formularios
- Interfaz responsive y atractiva
- Colores y tipografía consistentes

## Requisitos

- Flutter SDK (versión 3.0.0 o superior)
- Dart SDK
- Android Studio o VS Code
- Dispositivo móvil o emulador

## Instalación y Ejecución

1. **Clonar o descargar el proyecto**

   ```bash
   cd TallerMovil_Proyecto
   ```

2. **Instalar dependencias**

   ```bash
   flutter pub get
   ```

3. **Ejecutar la aplicación**
   ```bash
   flutter run
   ```

## Estructura del Proyecto

```
lib/
├── main.dart          # Punto de entrada de la aplicación
└── login_screen.dart  # Pantalla de login
```
